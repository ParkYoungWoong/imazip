# Imazip

'Imazip'은 프론트엔드 작업을 위해 이미지 크기와 품질을 설정할 수 있는 자바스크립트 모듈입니다.

## 설치

```
npm install imazip
```

## Options

### width

- Type: `Number`,
- Default: `null`

### height

- Type: `Number`,
- Default: `null`

### src

- Type: `String`,
- Default: `""`,
- Required!

### quality

- Type: `Number`,
- Default: `0.9`

### format

- Type: `String`,
- Default: `"image/png"`

### maxSize

- Type: `Number`,
- Default: `1920`

### thumbnail

- Type: `Object`,
- Default: `null`

### thumbnail.width

- Type: `Number`,
- Default: `null`

### thumbnail.height

- Type: `Number`,
- Default: `null`

### thumbnail.maxSize

- Type: `Number`,
- Default: `200`

### thumbnail.quality

- Type: `Number`,
- Default: `0.9`
